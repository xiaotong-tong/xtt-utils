# [randomList](../../../src//random//randomlist.ts)

返回一个随机数列表

## params

-   `min` (number)：最小值
-   `max` (number)：最大值
-   `count` (number)：随机数个数

## returns

-   `number[]`：随机数列表

## example

```js
randomList(1, 10, 5); // [8, 9, 10, 8, 10]
randomList(1, 10, 10); // [3, 5, 4, 1, 10, 7, 6, 9, 4, 10]
```
